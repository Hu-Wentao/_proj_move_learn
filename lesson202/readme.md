## NFT / Aptos Digital Assets Standard(TokenV2)
非同质化代币, 最早出现在 EIP-721
NFT常见的属性
- 名称
- 描述
- URI :外部链接 (图片的URL等)
// 如果NFT是一组集合, 那么还常有如下属性
- 供应: 集合中的总数量
- 最大值: mint最大值

### NFT扩展协议:
NFT的扩展, 可以用于表示可堆叠的资产, 如游戏中的木材资源
出现在 EIP-1155

## FA / Aptos Fungible Asset Standard (CoinV2) 
可互换资产标准, 它使得各种资产(大宗商品,房地产,金融工具等)可以Token化
